# Notifications

Кнопка — интерактивный элемент, на который можно нажать, который подразумевает какое-то действие.
Вы можете разместить текст и значки внутри кнопки.
Кнопки часто используются для отправки форм и для переключения элементов в режим просмотра.


## Notifications

Существует 3 варианта кнопок - `primary`,`gray` и `danger`
